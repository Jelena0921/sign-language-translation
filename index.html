Welcome!

This a real-time sign language translation prototype.

Instructions:

  • Allow access to webcam and microphone.
  • Train the gestures using the interface on the right, please make sure to train around 40 to 80 frames.
  • For the "_BASE_", please be still with no gestures, to allow the system learn your resting position.
  • For the "_PAUSE_", please cover the camera with your hand - this will allow pausing prediction when required. Alternative way to pause sign translation is clicking on the video.
  • Please make sure the gestures are static (require minimal movement) and are sufficiently different from each other.
  • Once complete, press on the button below.
